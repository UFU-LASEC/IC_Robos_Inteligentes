# 🤖 Desenvolvimento de Robôs Inteligentes: do Sumô Autônomo ao Controle via Aplicativo — LASEC

Este repositório reúne a documentação do projeto de pesquisa desenvolvido por Vitor Henrique para o LASEC (Laboratório de Automação, Sistemas Eletrônicos e Controle) voltado à robótica educacional, sistemas embarcados e aplicações de controle utilizando plataformas microcontroladas.

Os projetos têm como objetivo aplicar conceitos de programação, eletrônica, automação e robótica por meio do desenvolvimento prático de robôs autônomos e controlados remotamente.

---

# 🚀 Projetos

## 🥊 Robô Sumô

Robô desenvolvido para competições de sumô autônomo, utilizando sensores para detecção de adversários e estratégias de movimentação dentro da arena.

### Características
- Detecção de oponentes
- Controle de motores DC
- Estratégias de ataque
- Estrutura compacta
- Plataforma Arduino

---

## 🚧 Robô Desvia Obstáculos

Sistema robótico autônomo capaz de identificar e evitar obstáculos utilizando sensores de distância.

### Características
- Navegação autônoma
- Sensor ultrassônico
- Controle de direção
- Algoritmo de desvio
- Monitoramento em tempo real

---

## 📱 Robô Controlado via Aplicativo

Robô controlado remotamente por smartphone através de comunicação Bluetooth.

### Características
- Controle em tempo real
- Comunicação Bluetooth
- Aplicativo mobile
- Controle de velocidade e direção
- Interface simples e intuitiva

---

# 🛠️ Tecnologias Utilizadas

- Arduino
- C/C++
- Sensores Ultrassônicos
- Bluetooth HC-05 / HC-06
- Motores DC
- Ponte H
- Eletrônica Embarcada

---
